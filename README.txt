FOOD REPICES CLI

# What is this application ?
This is NodeJS application build with Spoonacular API. 
Currently, it help people to find recipes and enjoy theirs meals.


# Let's get started !
Install FOOD RECIPES CLI with "node index.js" : 
* --cook to search recipes
* --recipes to see all existing recipes
* --explications to see readme
* --author to see author of this application

# Dependencies : 
Commander
Inquirer
SQLITE
Unirest
